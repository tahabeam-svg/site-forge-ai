Build a complete production-ready SaaS platform for AI-powered website creation.

The platform should be inspired by the concept of Replit but simplified for non-technical users.

Users should be able to create, edit, preview, and publish websites using natural language without coding.

The platform must support Arabic as the primary interface language (RTL) and English as the secondary language (LTR).

The entire experience should allow users to build a professional website in less than two minutes.

Core Concept

Users type a description such as:

Create a website for a perfume exhibition in Riyadh.

or

أنشئ موقع شركة عطور في الرياض

The AI automatically generates:

• homepage
• about section
• services section
• gallery
• contact page
• responsive layout
• SEO metadata
• color palette
• icons
• images

No coding knowledge is required.

AI Chat Website Builder

The platform must include a conversational AI builder using:

OpenAI

Users can build and edit the website through chat commands.

Examples:
